# PHP Algorithm Training Project

**〜Paiza Bランクをサクッと解ける実力を身につけるための訓練プロジェクト〜**

## 🎯 目的

このプロジェクトは、PHPを使ってアルゴリズムの基礎力を高め、
**Paiza の B ランク問題を安定して解ける実力**を身につけることを目的とします。

特に、以下のスキルを重点強化します：

* 入力処理（標準入力）
* 配列処理
* 文字列処理
* シミュレーション問題
* 条件分岐・ループ
* 実装力（要件の整理 → コード化）
* 時間計算量の基礎（探索・ソート）
* よくあるアルゴリズム（累積和 / 2D 配列処理 / グリッド探索）
